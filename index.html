Đoạn mở đầu — mô tả nhanh. Bên dưới là một ô hiển thị mã nguồn/console, nhưng phần nội dung xung quanh vẫn hiển thị bình thường.

Ví dụ: tính giai thừa (Python)
Copied!
x = int(input("Nhập số cần tính giai thừa: "))

def fact(x):
    if x == 0:
        return 1
    return x * fact(x - 1)

print(fact(x))

Đoạn nội dung tiếp theo — bạn có thể chèn nhiều ô console khác hoặc thêm ảnh, biểu đồ... mà không làm hỏng layout.
